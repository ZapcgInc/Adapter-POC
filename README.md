**Agoda Adapter**

A HTTP API for connecting to Agoda Partner API.

>Tech Stack :

``
    
    Language : Scala (2.12.8)
    
    Framework : Finagle (https://twitter.github.io/finagle/)
``

>Set up :

``
    
    On OSX (Using homebrew):
    
    brew install scala
    
    brew install sbt
``

>To run the server:

``
    sbt run
``
